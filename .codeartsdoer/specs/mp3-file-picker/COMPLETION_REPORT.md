## 使用说明

### 用户操作流程
1. 打开音乐应用
2. 进入音乐列表页面
3. 点击"选择本地文件"按钮
4. 在文件选择器中选择MP3文件
5. 文件自动添加到播放列表
6. 点击文件列表中的播放按钮播放

### 注意事项
- 首次使用需要授权文件读取权限
- 支持的文件格式：MP3
- 最近列表最多保存20个文件
- 文件列表会按播放时间排序
